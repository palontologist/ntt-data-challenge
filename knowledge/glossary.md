# Data Astel Glossary

- **DA-Core**: The primary data processing engine used in all client projects.
- **M-Sprints**: Modified Agile Sprints specific to consulting delivery.
- **R-Metric**: Reliability Metric used to evaluate data quality.
- **T-ID**: Task Identifier (e.g., T-101).
- **B-Basis**: Baseline evidence gathered during the discovery phase.
